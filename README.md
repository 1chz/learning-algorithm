# Algorithm & Data Structure

* 브랜치명은 자신의 이름으로 한다 (ex. Changhun-Han)
* 자신의 브랜치에만 커밋한다
* 1문제당 1커밋을 원칙으로 한다
* 커밋 메시지는 간결하고 명확하게 작성한다

```
feat(<branch>): [<플랫폼>, <문제번호>] <문제이름>

<문제 링크>
``` 

# Example Commit Message
```
feat(changhun): [BOJ, 2557] Hello World

https://www.acmicpc.net/problem/2557
```
